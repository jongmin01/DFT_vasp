# Scripts Directory

Utility scripts for VASP calculations, analysis, and visualization.

## structure_generation/
Scripts for creating initial atomic structures
- Generate twisted heterostructures
- Create supercells
- Modify atomic positions

## analysis/
Data analysis scripts
- Extract data from vasprun.xml
- Calculate properties
- Process VASP output files

## plotting/
Visualization scripts
- Band structure plots
- DOS plots
- Structure visualization
- Data comparison plots
