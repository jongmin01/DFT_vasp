# TMDC Janus Heterostructure

## Project Description
Study of Janus transition metal dichalcogenide (TMDC) heterostructures.

## Current Research

### twisted_angle/
Investigation of twist angle effects on electronic properties of Janus TMDC heterostructures.

**Systems studied**: MoSSe/WSSe heterostructures

**Twist angles**: 0°, 10°, 20°, 30°, 40°, 50°, 60°

**Status**: In progress - 0° relaxation ongoing

## Directory Structure

Each angle directory contains:
- `relax/` - Structural optimization for the twisted configuration
- `scf/` - Self-consistent field calculation
- `band/` - Band structure calculation
- `dos/` - Density of states calculation

## Notes
- Focus on how twist angle affects band alignment and electronic coupling
- Each configuration requires careful structural relaxation
